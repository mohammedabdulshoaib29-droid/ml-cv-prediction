## 🚀 How to Get Started

### Step 1: Backend (Terminal 1)
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Step 2: Frontend (Terminal 2)
```bash
cd frontend
npm install
npm start
```

### Step 3: Use Application
```
Open: http://localhost:3000
```

**Total time: ~20 minutes**
