🐛 QUICK FIXES

Issue                         Solution
─────                         ────────
Port 8000 in use    →  netstat -ano | findstr :8000
                       taskkill /PID <PID> /F

Port 3000 in use    →  netstat -ano | findstr :3000
                       taskkill /PID <PID> /F

Module not found    →  pip install -r requirements.txt

npm issues          →  Delete node_modules, run npm install

CORS error          →  Check backend running on port 8000

No datasets         →  python backend/datasets/generate...

Blank page          →  browser F12 → Console tab
